## 🌊 About the Website

This portfolio evolved from a static design into a **dynamic full-stack project**, combining:

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Node.js, Express
* **External Services:** Formspree (contact handling)

The visual theme reflects:

* 🌊 glowing underwater atmosphere
* 🧠 modern minimalism
* 🦁 strength, elegance, and personal identity

---

## ✨ Features

### 🔍 Smart Search System

* Intelligent search across sections
* Partial matching & keyword suggestions
* Smooth scroll navigation to results

### 📊 Visitor Analytics (Custom Backend)

* Built with **Node.js + Express**
* Tracks visits and stores them
* Admin panel to view analytics

### 📩 Contact System

* Integrated **Formspree** for global message handling
* No backend dependency for contact
* Works even when server is offline

### 🦉 Dynamic UI Enhancements

* Animated UI elements (including owl companion 🦉)
* Light/Dark mode behavior
* Interactive and responsive design

### 🔐 Admin Panel

* Protected access via admin key
* Displays visitor statistics
* Backend validation for security

### 📱 Responsive Design

* Fully optimized for:

  * Desktop
  * Tablets
  * Mobile devices
